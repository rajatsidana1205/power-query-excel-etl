# Power Query ETL using Excel

## Overview
This project demonstrates an automated data cleaning and transformation workflow built using Power Query in Excel.

## Tools Used
- Microsoft Excel
- Power Query (M language)

## Key Transformations
- Cleaned and structured raw order data
- Removed unnecessary and null columns
- Converted data types for dates and numeric fields
- Created derived metrics such as shipping duration
- Prepared analysis-ready datasets for reporting
- Built summary-level insights using transformed data

## Outcome
The solution enables one-click refresh, reduces manual data preparation effort, and produces clean datasets suitable for business analysis and reporting.
